# Pokedle

Juego estilo wordle sobre Pokémon , contiene los siguientes modos de juego:
- Región de Kanto
- Región de Johto
- Región de Hoenh
- Región de Sinnoh
- Región de Teselia
- Región de Kalos
- Región de Alola
- Región de Galar
- Región de Paldea
- Pokedex completa
 App desarrollada en Java por Javier Bautista Bonilla
